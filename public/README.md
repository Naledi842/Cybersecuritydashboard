# Cyber Security Authentication & Monitoring Dashboard

## Overview
This project is a Cyber Security login and monitoring dashboard that simulates a secure authentication system and tracks login activity. The system records login attempts and displays them in a security dashboard to demonstrate how authentication monitoring works in cybersecurity environments.

## Features
- Secure login page
- Login attempt tracking
- Session duration monitoring
- Login history logging
- Cyber security dashboard interface
- Neon themed UI (Purple / Blue / Black)

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- SQLite Database

## How It Works
1. A user enters their email and password on the login page.
2. The server verifies the credentials.
3. Each login attempt is recorded in the database.
4. Successful logins redirect the user to the security dashboard.
5. The dashboard displays login history and session information.
